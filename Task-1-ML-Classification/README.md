# Task 1: IPL Match Prediction (Classification)

## Project Overview
This project involves building a supervised machine learning model to predict the outcome of IPL matches. It compares two popular algorithms to determine which provides better accuracy and reliability for sports data prediction.

## Goal
To build and evaluate a classification model that predicts if the home team will win based on features like team form, toss results, and rest days.

## Requirements Covered
* **Data Preprocessing:** Handled scaling and train/test splits.
* **Algorithms Compared:** Logistic Regression and Random Forest.
* **Metrics Reported:** Accuracy, Precision, Recall, F1-score, and ROC-AUC.

## Files in this Folder
* `intership1task.ipynb`: The main Jupyter Notebook containing the code, data generation, and visualizations.
* `IPL_Match_Prediction_Report.pdf`: A formal report summarizing the findings and model performance.

## Results
The models were evaluated using 5-fold cross-validation. The Random Forest model provided a robust ensemble approach, while Logistic Regression identified the linear relationship between form and match outcome.
